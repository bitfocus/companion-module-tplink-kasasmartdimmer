# TP-Link Kasa Smart Dimmer Switch

This module controls TP-Link Kasa Smart Dimmer Switches.

Actions:

* Power On/Off/Toggle
* Set Brightness (0-100)
* Brightness Up/Down

Variables:

* SW Version
* HW Version
* Model
* Device ID
* OEM ID
* HW ID
* RSSI
* Latitude
* Longitude
* Alias
* MAC Address
* Power State
* Brightness Level

Feedbacks:

* Power State

Presets:

* Power On/Off/Toggle
* Brightness Up/Down
* Brightness in 10% Increments
